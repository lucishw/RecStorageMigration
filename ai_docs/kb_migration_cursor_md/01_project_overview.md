# 프로젝트 개요

## 프로젝트명

KB손보 파일 마이그레이션 프로그램

## 목적

DB 기준으로 마이그레이션 대상 파일을 조회한 뒤, 기존 스토리지의 파일을 TOBE 신규 스토리지로 복제한다.

## 개발 방향

- .NET 8 기반 신규 프로젝트로 구성한다.
- 기존 프로그램을 일부 수정하는 방식이 아니라 신규 구조로 재설계한다.
- 기존에 프로그램 여러 개를 띄워 병렬 처리하던 구조를 사용하지 않는다.
- 하나의 프로그램 내부에서 여러 Worker를 실행하여 병렬 처리한다.
- DB 조회와 DB 업데이트는 병목이 되지 않도록 Batch/Bulk 구조로 설계한다.
- 파일 복제 처리 중 Network Bandwidth 사용량을 제어한다.
- UI는 단순 작업 화면이 아니라 실시간 운영 대시보드 형태로 구성한다.

## 주요 사용자

- 운영자
- 데이터 마이그레이션 담당 엔지니어
- 장애 대응 담당자
- 개발자

## 핵심 기능

- DB 기준 대상 파일 조회
- 기존 스토리지 파일 존재 여부 확인
- TOBE 스토리지로 파일 복제
- Worker 병렬 처리
- 처리 결과 Bulk Update
- 네트워크 대역폭 제어
- 실시간 대시보드
- 오류 로그 및 실패 파일 관리
- 재시도 및 재처리
- 검증용 테스트 데이터 생성
- 기능/성능 검증 시나리오 지원
